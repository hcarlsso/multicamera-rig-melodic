264 /*
265 Packet Overview
266 
267 Byte
268 0 Packet start byte
269 1 Packet indo: ID (mask 0x1F) | reserved bits (mask 0xE)
270 2 Packet counter (for ACK and retry)
271 3 Packet flags
272 
273 // packet body, may or may not exist depending on packet id - packet body is made up of 4 byte or 8 byte values.
274 4-7 Data identifier
275 8-11 Data length
276 12-15 Data offset
277 16-19 Data start
278 (n-8)-(n-5) Last piece of data
279 // end data
280 
281 n-4 Reserved
282 n-3 Checksum high byte
283 n-2 Checksum low byte
284 n-1 Packet end byte
285 */
